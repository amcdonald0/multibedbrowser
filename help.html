Uploading Files

Follow these steps to upload your genomic data files:

  1. Go to the Upload page.
  2. Enter a file name and description.
  3. Select a project from the dropdown menu.
  4. Choose a file to upload (supported formats: .bed, .csv,.txt).
  5. Click the Upload button.

After uploading, you can view and manage your files in the "View Files" section. Ensure your file type and size are supported.

Successful Upload: You have successfully uploaded your file.
Error Message: Upload failed. Please check file type, size, and project selection.

Searching Files

Use the Search page to analyze uploaded data. Filter your results by project, query type, or genomic region. Results are displayed in a table format.

  1. Go to the Search page.
  2. Select a BEDTool.
  3. Choose the Project and File names.
  4. Click Run Analysis to view results.

Bedtools Utilities

The following table lists all the Bedtools commands integrated into the platform:

Command Description
Intersect Finds overlapping intervals between two datasets, allowing fine control over how intersections are reported.
Merge Combines overlapping or adjacent intervals into single intervals, simplifying datasets and reducing redundancy.
Subtract Removes overlapping intervals between two datasets, isolating unique regions in a dataset.
Coverage Calculates the depth of coverage for features in a dataset, useful for understanding sequencing data representation.
Genomecov Generates genome-wide coverage summaries, which are essential for visualizing sequencing depth and identifying regions of high or low coverage.
Closest Identifies the nearest feature in one dataset to each feature in another dataset, helping associate genomic elements like genes and regulatory regions.
Getfasta Extracts sequences from a reference genome based on input BED coordinates, enabling retrieval of specific genomic regions for further analysis.
Shuffle Randomizes the positions of features within a genome while preserving constraints, useful for creating null models in statistical analyses.
Map Aggregates values from one dataset onto features of another dataset, summarizing scores or annotations across genomic intervals.
Groupby Performs operations like summing or averaging on grouped data based on specified columns, helpful for summarizing complex datasets.
Slop Expands or contracts genomic intervals by a specified number of bases, often used to adjust regions for motif searches or other analyses.
AnnotateBed Annotates genomic regions with information from other datasets, often used to integrate functional annotations with genomic intervals.
Multicov Calculates coverage across multiple datasets simultaneously, providing a comprehensive view of sequencing data overlap.
BamToBed Converts BAM files into BED format, enabling downstream analysis with other Bedtools utilities.
Complement Identifies regions of the genome not covered by input intervals, useful for finding gaps or unannotated regions in datasets.
Overlap Quantifies or identifies the degree of overlap between features in two datasets, e.g., calculating percent overlap.
