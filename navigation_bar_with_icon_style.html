Uploading Files

Follow these steps to upload your genomic data files:

  1. Go to the Upload page.
  2. Enter a file name and description.
  3. Select a project from the dropdown menu.
  4. Choose a file to upload (supported formats: .bed, .csv, .txt).
  5. Click the Upload button.

After uploading, you can view and manage your files in the "View Files" section. Ensure your file type and size are supported.

Successful Upload: You have successfully uploaded your file.
Error Message: Upload failed. Please check file type, size, and project selection.
